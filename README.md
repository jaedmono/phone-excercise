# Getting Started

### How to build this project

To build and generate the jar file just need to run the command below

`./gradlew clean build `

### How to run this project

This project is using the sprint boot framework and to persist 
the data it was used and H2 Database, saying that  to run the project is 
just to execute the command below

`./gradlew bootRun`
